# Changelog - Weed JS Mod

All notable changes to this project in 1.1.0 version will be documented in this file.

---

## Features

- **Changes in Cannabis Nugget** — Variants Common, Rare, Special y Legendary dropped with probability
- **Changes in Blunt Wraps** — Blunt wraps now are crafted in the cannabis crafting table and required 1x Paper and 1x flavor (Leaves, Apple, Watermelon, Honey, Cocoa, Gold block)
- **Changes in Blunt** — Blunt has components that affect effects and their duration
- **New Tool** — Grinder to process unprocessed nuggets
- **New Crafting Table** — Cannabis Crafting Table to craft blunt wraps and blunts
- **New Soil** — Only block able to plant cannabis seed
- **Seed craft eliminated** — Only obtainable via trades
- **New Grow Light** — Block that lights and accelerate grow and improve quality
- **New Villager Profession** — Related to cannabis crafting table

---

## Crafting

### Cannabis Crafting Table
Crafted in a shaped 3x3 pattern:
```
[Oak Plank] [Iron Ingot]    [Oak Plank]
[Redstone]  [Green Dye]     [Redstone]
[Oak Plank] [Oak Plank]     [Oak Plank]
```

### Cannabis Soil
Crafted in a shapeless pattern:
- 1× Brown Mushroom
- 1× Dirt
- 1× Moss Block
- 1× Bone Meal

### Grow Light
Crafted in a shaped 3x3 pattern:
```
[Glass]     [Glass]         [Glass]
[Redstone]  [Purple Dye]    [Redstone]
[Glass]     [Redstone]      [Glass]
```

### Grinder
Crafted in a shaped 2x3 pattern:
```
[Iron Nugget]   [Iron Nugget]   [Iron Nugget]
[Iron Ingot]    [Iron Ingot]    [Iron Ingot]
```
