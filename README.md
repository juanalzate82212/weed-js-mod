# Weed JS Mod

A Fabric mod for Minecraft 1.21.11 that adds a full cannabis cultivation cycle to your world — grow it, harvest it, roll it, smoke it.

---

## Features

- **New crop** — Cannabis plant with 8 growth stages and cross rendering
- **Full crafting cycle** — From basic ingredients to a smokeable consumable
- **Interactive consumable** — Requires a lighter in your offhand to use
- **Visual effects** — Campfire smoke particles when smoking
- **Status effect** — Nausea effect after consuming
- **Advancements** — 4 custom advancements tracking your progression
- **Creative tab** — All mod items organized in their own tab
- **Multilingual** — English, Spanish (ES) and Spanish (MX) supported

---

## Items

| Item | Description |
|------|-------------|
| Cannabis Seed | Crafteable from basic ingredients. Plant it on farmland. |
| Cannabis Nugget | Dropped when harvesting a fully grown cannabis plant. |
| Blunt Wrap | Crafted from base game items. |
| Blunt | The final consumable. Combine a nugget and a wrap to craft it. |

---

## Crafting

### Cannabis Seed
Crafted in a shaped 3×3 pattern:

```
[Red Mushroom] [Green Dye]    [Red Mushroom]
[Green Dye]    [Wheat Seeds]  [Green Dye]
[Brown Mushroom][Green Dye]  [Brown Mushroom]
```

### Blunt
Crafted in a shapeless pattern:
- 1× Cannabis Nugget
- 1× Blunt Wrap

---

## Cultivation Cycle

1. **Craft** a Cannabis Seed using the recipe above
2. **Plant** it on farmland — you'll get a Glass Bottle back
3. **Wait** for it to grow through 8 stages
4. **Harvest** when fully grown — drops 1 Cannabis Nugget
5. **Craft** a Blunt Wrap from base game items
6. **Combine** the nugget and wrap to craft a Blunt
7. **Smoke** it — hold it in your main hand with a Flint & Steel in your offhand, then right-click and hold
